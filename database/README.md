# 咖啡点餐管理系统数据库说明

## 数据库配置
- 数据库名称：`coffee_ordering_system`
- 字符集：`utf8mb4`
- 排序规则：`utf8mb4_unicode_ci`

## 文件说明
- `schema.sql` - 数据库结构和基本数据
- `init_data.sql` - 初始化测试数据
- `README.md` - 数据库说明文档

## 数据库表结构

### 用户表 (users)
存储用户基本信息，包括用户名、邮箱、密码、角色等。

### 菜单表 (menu)
存储咖啡店菜单项信息，包括名称、描述、价格、分类等。

### 订单表 (orders)
存储订单信息，包括订单号、用户ID、总价、状态等。

### 订单明细表 (order_items)
存储每个订单的具体商品信息，关联菜单和订单。

## 数据库特性
- 支持用户角色管理（管理员/普通用户）
- 菜单分类管理
- 订单状态跟踪
- 订单详情视图
- 订单号自动生成
- 完整的外键约束
- 索引优化查询性能
