## 🛠️ 技术栈总结

### 后端技术
- **框架**: Flask 2.3.3
- **数据库**: MySQL + SQLAlchemy ORM
- **认证**: Flask-JWT-Extended
- **验证**: Flask-Marshmallow
- **密码加密**: Flask-Bcrypt
- **跨域**: Flask-CORS
- **文件上传**: Werkzeug

### 前端技术
- **框架**: Vue 3.3.4
- **构建工具**: Vite 4.4.9
- **UI组件**: Element Plus 2.3.9
- **状态管理**: Pinia 2.1.6
- **路由**: Vue Router 4.2.4
- **HTTP客户端**: Axios 1.5.0
- **动画**: Animate.css 4.1.1
- **样式**: SCSS/PostCSS

### 开发工具
- **包管理**: npm
- **代码规范**: ESLint + Prettier
- **组件自动导入**: unplugin-vue-components
- **图标**: Element Plus Icons
